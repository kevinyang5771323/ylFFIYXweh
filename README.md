# 台球厅管理系统简介 java725

## 项目概述

台球厅管理系统是基于SSM（Spring + SpringMvc + Mybatis）和Vue的前后端分离项目。采用MVC三层体系结构，利用Maven进行项目管理和构建。系统实现了基于B/S结构的Web应用，界面简洁，操作便捷。

## 技术栈

- 前端：Vue
- 后端：Java EE
- 框架：Spring, SpringMvc, Mybatis
- 数据库：MySQL
- 项目管理：Maven
